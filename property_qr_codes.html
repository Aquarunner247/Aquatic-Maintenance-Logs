  1. Print the QR codes: Click the "Print QR Codes" button below to print all codes.
  2. Place at properties: Post each QR code at its corresponding property location (near the meters is ideal).
  3. Scan to enter readings: Technicians scan the QR code with their phone to access the form for that specific property.
  4. Submit readings: Fill in water, electric, and gas meter readings along with any notes.
  5. View dashboard: Access the dashboard to see all readings across all properties.

Note: All data is stored locally in the browser. For production use, consider hosting this on a web server with a proper database backend. You can customize the property names and add more properties by editing the HTML file.
